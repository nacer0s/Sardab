# Sardab — سرداب

> تواصل مشفر بالكامل • ند لند • بدون خوادم

**سربد** منصة تواصل لامركزية تتيح لك إرسال الرسائل والملفات وإجراء المكالمات الصوتية والمرئية والاجتماعات الجماعية — كل ذلك بتشفير تام من طرف إلى طرف دون المرور عبر أي خادم.

🔗 **الموقع الرسمي:** [https://sardab.gt.tc](https://sardab.gt.tc)

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| 💬 **المراسلة الفورية** | دردشة مباشرة مشفرة AES-256-GCM |
| 📁 **إرسال الملفات** | إرسال ملفات مشفرة حتى 256 ميغابايت |
| 🔐 **الخزنة النصية** | تخزين نصوص مشفرة مع حرق بعد القراءة |
| 📞 **المكالمات الصوتية** | مكالمات صوتية P2P مشفرة |
| 🎥 **المكالمات المرئية** | فيديو مباشر مشفر مع صورة داخل الصورة |
| 👥 **الاجتماعات الجماعية** | اجتماعات جماعية بتقنية Mesh P2P |
| 🖥️ **مشاركة الشاشة** | مشاركة الشاشة خلال الاجتماعات |

## 🛡️ الأمان

- **تشفير AES-256-GCM** عبر Web Crypto API
- **PBKDF2 بـ 600,000 تكرار** لاشتقاق المفتاح
- **اتصال P2P مباشر** — الخادم لا يرى محتوى الرسائل أو المفاتيح أبدًا
- **حرق تلقائي** للرسائل والملفات بعد قطع الاتصال
- **بدون حسابات** — لا بريد إلكتروني ولا رقم هاتف

## 🧱 التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| PHP 8.0+ | الخادم الخلفي (توجيه WebRTC، الخزنة النصية) |
| Apache (.htaccess) | التوجيه، CSP، الأمان |
| WebRTC API | الاتصال المباشر P2P |
| Web Crypto API | التشفير من طرف إلى طرف |
| HTML5 / CSS3 | واجهة المستخدم |
| JavaScript (ES6) | منطق التطبيق والمتصفح |

## 🖥️ التثبيت المحلي (للتطوير فقط)

> **ملاحظة:** هذا مسموح به فقط للتطوير والتدقيق. لا يُسمح بتشغيل نسخة عامة من هذا المشروع.

```bash
# 1. انسخ المشروع إلى localhost
# 2. تأكد من تشغيل PHP 8.0+ و Apache مع mod_rewrite
# 3. شغل HTTPS محليًا (مطلوب لتشفير WebRTC و Web Crypto)
# 4. افتح المتصفح على https://localhost/Sardab
```

**متطلبات التشغيل المحلي:**
- PHP 8.0+
- Apache مع mod_rewrite
- HTTPS (للـ WebRTC و Web Crypto API)
- متصفح حديث (Chrome، Firefox، Safari، Edge)
